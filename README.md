## Installation
1. Install dependencies using Pipenv:
    ```sh
    pipenv install
    ```
2. Create `.env` file with the following content:
    ```sh
    API_BASE_URL=
    USER_API_BASE_URL=
    USER_ID=
    USER_NAME=
    USER_EMAIL=
    USER_PASSWORD=
    DATABASE_NAME=
    DATABASE_USER=
    DATABASE_PASSWORD=
    DATABASE_HOST=
    DATABASE_PORT=
    UI_BASE_URL=
    GOOGLE_NAME=
    GOOGLE_EMAIL=
    GOOGLE_PASSWORD=
    MAILSLURP_BASE_URL=
    MAILSLURP_API_KEY=
    ```
